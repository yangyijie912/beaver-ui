# beaver-ui

个人用的组件库，React + TypeScript + tsup + Storybook。

快速开始

1. 安装依赖：`pnpm install`
2. 启动 Storybook：`pnpm storybook`
3. 构建包：`pnpm build`

目录说明

- `src/`：源码
- `src/components`：组件实现
- `src/tokens`：vanilla-extract tokens
- `.storybook/`：Storybook 配置

[文档参考(Docs/Storybook)](https://yangyijie912.github.io/beaver-ui)
